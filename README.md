🛡️ PhishGuard – AI-Based Phishing Link Detection Browser Extension

PhishGuard is an intelligent browser extension that detects phishing and malicious links in real time using AI.
It protects users from fake websites, credential-stealing attacks, and suspicious URLs before they cause harm.

🚀 Features

🔍 Real-Time URL Scanning
Analyzes links instantly when you visit or click them.

🤖 AI-Powered Detection
Uses machine learning models to identify phishing patterns in URLs.

⚠️ Instant Warning Alerts
Shows security alerts when a suspicious or malicious link is detected.

🌐 Works Across Websites
Protects you while browsing, searching, or opening links from emails or messages.

🧠 Smart URL Analysis

Domain analysis

Suspicious keyword detection

URL structure evaluation

Phishing pattern recognition

🔒 Privacy Friendly
No unnecessary user data collection.

🧠 How It Works

PhishGuard uses an AI/ML model trained on phishing and legitimate URL datasets.

The extension workflow:

User visits or clicks a link

Extension extracts the URL

URL is analyzed using the AI phishing detection model

Risk score is generated

If malicious → warning popup is displayed

User Clicks URL
      │
      ▼
Extension Captures Link
      │
      ▼
AI Model Analyzes URL
      │
      ▼
Safe ✅      Phishing ⚠️
Allow       Show Warning
